## Getting Started

First, run the development server:  

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

**If localhost doesn’t load (404, 500, or blank):**
- Only run **one** `npm run dev` at a time. Another app or an old Next dev server may be using port 3000 and returning 404.
- Stop any running dev servers (Ctrl+C in the terminal where `npm run dev` is running).
- Clear the dev cache and restart:
  ```bash
  rm -rf .next/dev
  npm run dev
  ```
- Use **http://localhost:3000** (not https). If the terminal says “using port 3001”, open http://localhost:3001 instead.
- The first load can take a few seconds while Next compiles; wait for “Ready” then refresh if needed.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
